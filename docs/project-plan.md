# TradeFix E-Commerce Project Plan

## Project Overview

TradeFix is a modern B2B/B2C e-commerce platform for selling construction materials, power tools, hand tools, safety equipment, and building supplies.

The application is built using modern web technologies with a focus on:

- Security
- Performance
- Scalability
- Maintainability
- Clean Architecture
- Professional UI/UX

---

# Technology Stack

## Frontend

- Next.js 15 (App Router)
- React
- TypeScript
- Tailwind CSS

## Backend

- Next.js Server Actions
- Prisma ORM
- PostgreSQL

## Authentication

- Auth.js (NextAuth)
- bcrypt password hashing

## Payments

- Stripe

## Deployment

- Vercel
